# 🎲 Snake and Ladders Game

A simple Snake and Ladders game built using **HTML, CSS, and JavaScript**.

## Features
- 🎮 Two-player game
- 🎲 Dice roll animation
- 🐍 Snakes and 🪜 Ladders
- 📊 Player score and turn tracking
- 🔄 Reset game
- 📱 Responsive design

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project.
2. Open `snake and ladders.html` in your browser.
